## 📂 仓库内容

* `dayX.py`：每天的学习练习代码（例如 `day1.py`, `day2.py` 等）
* `my_note.txt`：学习过程中整理的笔记与心得
* `mini_projects/`：一些小工具或小项目，例如班次统计器、绘图工具等
* 其他：实验文件、测试代码、临时记录等

* ndarray、多维数组操作
* 数据可视化与绘图

### 3. PyTorch 入门 ⏳

* 自动求导与基本神经网络
